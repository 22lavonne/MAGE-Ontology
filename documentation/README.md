# Documentation Folder

This directory contains hand-maintained project documentation for different aspects of the ontology and knowledge graphs.

## Contents:
- `axiomization.md`: contains the list of all axioms in a table and another table with the axiom selection identifying each kind of axiom.
- `axioms.md`: contains a description of all the different modules of the schema, a list of their respective axioms, and a schema diagram of each module. It also contains the overall schema diagram image.
- `competency-questions-queries.md`: contains all the competency questions and their corresponding SPARQL queries.
- `key-notions.md`: outlines the key notions of the ontology, including any connected patterns and datasets associated with them.
- `use-case.md`: outlines the use case of the ontology, including the narrative and the list of competency questions.
- `schema`: includes the .graphml file of the schema and a subdirectory containing screenshots of all the schema modules.
